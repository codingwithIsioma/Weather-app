**Description:** I built the application using the Open-Meteo API to fetch the weather data of a specified location. It receives the user's input and processes and display's the information specific to that location. I also added features to toggle between celsius or fahrenheit, and using the in-built Javascript Geolocation API, to access a user's current location and display the weather information of that location.
